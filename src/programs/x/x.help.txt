x.js: Quick calculator.

Usage:
  x [args] [mode]

Basic functions:
  log, sqrt, abs, sin, cos, tan, ceil,
  floor, rand, min, max, round, sign

More functions:
  sum, prod, ints, hash, hash64

Useful constants:
  pi, e, c

Scaling constants:
  f, p, n, mu, m, k, M, G, B, T, P

Options:
  -s, --scientific
  -x, --hexadecimal
  -d, --decimal
  -q, --quit
  -h, --help

For more information and help:
  https://github.com/jwmza/nt/blob/main/x.md
